## 🎮 遊戲介紹

呢個係一個簡單嘅「真心話大冒險」網頁版遊戲，專為廣東話玩家而設！唔使再煩住諗題目，無論你係聚會、Party 定係宿舍傾計，都可以即刻用嚟玩，炒熱氣氛，玩到停唔到手！

## 🌟 主要功能

| 功能 | 描述 |
|------|------|
| ✅ 即時生成題目 | 真心話或者大冒險題目一撳即有 |
| ✅ 分類選擇 | 🎲 全部 / 💬 真心話 / 🔥 大冒險 三種模式 |
| ✅ 地道廣東話 | 170+ 條題目，夠貼地 |
| ✅ 跨平台 | 手機、電腦都玩得 |
| ✅ 無需安裝 | 打開網頁就可以玩 |
| ✅ 深色/淺色主題 | 一鍵切換，護眼又靚 |
| ✅ 自定義題目 | 加入你自己嘅問題 |
| ✅ 本地儲存 | 唔會冇咗啲題目 |
| ✅ 自動部署 | Push 到 main 即自動更新網站 |

## 💡 點玩？

1️⃣ **打開網頁** - 直接去 [遊戲頁面](https://heilcheng.github.io/truthordare)

2️⃣ **揀類別** - 選擇「全部」、「真心話」或「大冒險」

3️⃣ **撳「生成問題」** - 一撳就有新題目（仲有抽獎動畫！）

4️⃣ **即刻拎到題目** - 跟住照做 or 答問題

就咁簡單，任何場合都啱玩！

## 🎨 功能特色

### 🎯 分類模式
- **🎲 全部 (170題)** - 隨機抽取所有題目
- **💬 真心話 (130題)** - 只抽真心話問題
- **🔥 大冒險 (40題)** - 只抽大冒險挑戰

### 🌙 深色/淺色模式
- 簡潔黑白灰設計
- 手動切換深色/淺色模式
- 現代感 UI，唔會老土

### 📱 響應式設計
- 完美適配所有屏幕尺寸
- 橫屏/豎屏自動調整
- 手機體驗一流

### 💾 數據持久化
- 本地儲存自定義題目
- 唔會因為刷新頁面而冇咗

### 🎰 抽獎動畫
- 撳「生成問題」會有老虎機式動畫
- 睇住題目快速閃過，增加刺激感

## 🛠️ 技術棧

- **React 18** - 前端框架
- **Material-UI** - UI 組件庫
- **GitHub Actions** - 自動部署
- **GitHub Pages** - 網頁託管

## 🚀 本地開發

```bash
# Clone 項目
git clone https://github.com/heilcheng/truthordare.git
cd truthordare

# 安裝依賴
npm install

# 啟動開發伺服器
npm start

# 打包生產版本
npm run build
```

## 📦 自動部署

每次 push 到 `main` branch，GitHub Actions 會自動：
1. 安裝依賴
2. Build 項目
3. 部署到 GitHub Pages

唔使手動做任何嘢！
